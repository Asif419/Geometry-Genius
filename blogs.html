1. What is DOM and what is the purpose of DOM?

The full form of dom is ‘Document object Model’. And this is an interface of programming for web documents. Now there are many purposes of DOM. First of all, we can create, change or remove html elements through DOM. Furthermore, we can also modify elements of website’s any elements.

2. How will you select HTML elements using DOM? Name the DOM methods

Well there are five ways to select an element from website via DOM.
1. getElementsByTagName()
2. getElementsByClassName()
3. getElementById()
4. querySelector()
5. querySelectorAll()
All those selector are used most in DOM.

3. What is event bubble?

Event bubbling is a concept od Document Object Model (DOM). When an element receive an event and that event transmitted to it’s parents elements in the tree of DOM till root element of targeted element.
To prevent this we can use stopPropagation(). this method stops event to travel to it’s parents element

4. What is a callback function and why will you use it?

The simple definition of callback function is- ‘CallBack function is a function who can pass an argument of another function’. This function usually execute inside main function but main function normally decide when this function will execute.
For response of an event we can use callback function. Benefit of this function is we can wait for the previous function called.
